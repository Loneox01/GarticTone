# Welcome to GarticTone
Gartic Tone is a musical spin-off of Gartic Phone, a popular party web game.  
Visit the Live Website at: gartic-tone.vercel.app/

## UPDATED 02/23/26
- As of now, Gartic Tone has transitioned away from a python backend in favor of a lighter SQL database system (Supabase).
- The frontend is written in TypeScript/REACT deployed via Vercel.
- "Classic" is currently the only finished game mode, with "Blind Karaoke" being a Work In Progress.
